Uniform Resource Locator (URL)

URL illustration

Definition:

A URL (Uniform Resource Locator) is the address of a specific resource on the Internet, such as a webpage, image, or file.

It consists of several components, including the protocol (e.g., http, https), the domain name (e.g., www.example.com), and the path to the specific resource (e.g., /page1.html).
