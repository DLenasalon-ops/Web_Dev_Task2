This site is designed to help you explore and understand the key concepts that power the internet — from TCP/IP to Web 4.0. Whether you're a student or just curious, you'll find clear explanations and examples here.

Internet Terminology Overview

Explore the essential terms that form the foundation of the internet:

  • TCP/IP: The fundamental communication protocols of the internet.
  • Web Browsers: Software applications for accessing web content.
  • Web Servers: Computers that host websites and serve content to users.
  • URL: Uniform Resource Locator, the address of a web resource.
  • DNS: Domain Name System, translates domain names to IP addresses.
  • HTTP: Hypertext Transfer Protocol, the protocol for transferring web pages.
  • Web 1.0 to Web 4.0: The evolution of the web from static pages to intelligent, immersive experiences.
  • Multi-tier Architecture: A software architecture pattern that separates an application into layers.
  • FTP: File Transfer Protocol, used for transferring files over the internet.
  • Intranet: A private network accessible only to an organization's members.
  • Extranet: A controlled private network allowing access to partners or customers.
