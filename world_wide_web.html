World Wide Web

WWW Illustration

Definition:

The World Wide Web (WWW) is an information system where documents and other web resources are identified by URLs, interlinked by hypertext links, and accessed via the Internet.

History:

Invented in 1989 by Tim Berners-Lee at CERN, the first website went live in 1991. The WWW quickly became the backbone of global communication, commerce, and education.

How It Works:

  • Client_Server Model: Browsers request resources from servers.
  • HTTP Protocol: Defines how data is transferred.
  • HTML: Structures documents with text, images, and links.
  • URLs: Provide unique addresses for resources.

Types of Web Content:

  • Surface Web: Indexed by search engines.
  • Deep Web: Includes databases and private content not indexed.
  • Dark Web: Encrypted networks accessible with special tools.

Evolution:

  • Web 1.0: Static pages, read-only.
  • Web 2.0: Interactive, user-generated content.
  • Web 3.0: Semantic web, AI, blockchain.
  • Web 4.0: Emerging intelligent, autonomous systems.
